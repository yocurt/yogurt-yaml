=== Bracket

ID[::performance::syntax_bracket]

==== Example evaluated

All proper yaml is allowed

```
ID#[test, content: [A, "]]]}{[[]}}"]]
```

=== Closure

ID[::performance::syntax_closure]


==== Example unevaluated

Everything according to yaml, exept the chars '{' and '}' is allowd.

Allowed:

```
ID#{test, content: [A, "]]]"]}
```

Not allowed:

```
ID#{test, content: [{A: abc}, "{} }"]}
```
